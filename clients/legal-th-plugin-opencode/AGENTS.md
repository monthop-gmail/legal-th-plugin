# Legal-TH Plugin

AI-powered legal productivity plugin สำหรับกฎหมายไทย

## ภาพรวม

Plugin นี้ช่วยทีมกฎหมายในไทยด้วยความสามารถหลัก:
- รีวิวสัญญาตาม playbook ขององค์กร (GREEN/YELLOW/RED classification)
- ตรวจสอบ compliance ตามกฎหมายไทย (PDPA, แรงงาน, แพ่ง ฯลฯ)
- คัดกรอง NDA อย่างรวดเร็ว
- ค้นหากฎหมายไทยจากฐานข้อมูล (ผ่าน MCP server)
- ศัพท์กฎหมายไทย-อังกฤษ
- สรุปข้อมูลกฎหมาย (รายวัน/หัวข้อ/เหตุการณ์)

## MCP Server

Plugin เชื่อมต่อกับ Legal-TH MCP Server สำหรับ:
- ฐานข้อมูลกฎหมายไทย (full-text search)
- Template สัญญา
- Compliance engine
- Glossary ศัพท์กฎหมาย
- Regulatory updates

## Playbook

องค์กรสามารถ config จุดยืนในการรีวิวสัญญาได้ที่ `legal.local.md`

## Disclaimer

Plugin นี้ช่วยงาน workflow ทางกฎหมาย ไม่ใช่คำปรึกษาทางกฎหมาย ผลวิเคราะห์ต้องได้รับการตรวจสอบจากทนายความที่มีคุณสมบัติ
