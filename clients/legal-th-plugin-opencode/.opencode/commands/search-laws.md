# /search-laws -- ค้นหากฎหมายไทย

> หากเห็น placeholder ที่ไม่คุ้นเคย ดู [CONNECTORS.md](../CONNECTORS.md)

ค้นหากฎหมาย ระเบียบ ข้อบังคับ และคำพิพากษาไทยจากฐานข้อมูลผ่าน ~~legal-th

**สำคัญ**: ผลการค้นหาเป็นข้อมูลอ้างอิง ไม่ใช่คำปรึกษาทางกฎหมาย ควรตรวจสอบกับแหล่งข้อมูลทางการ

## Usage

```
/search-laws $ARGUMENTS
```

ตัวอย่าง:
- `/search-laws PDPA การส่งข้อมูลข้ามประเทศ`
- `/search-laws พ.ร.บ.คุ้มครองแรงงาน การเลิกจ้าง`
- `/search-laws สัญญาเช่า อสังหาริมทรัพย์`
- `/search-laws ภาษีหัก ณ ที่จ่าย บริการ`

## Workflow

### ขั้นตอน 1: รับคำค้นหา

รับคำค้นหาจากผู้ใช้ ถ้าคลุมเครือ ถามเพิ่ม:
- **หัวข้ออะไร**: ระบุให้ชัดเจน
- **บริบท**: ทำไมถึงต้องการค้นหา
- **ขอบเขต**: ต้องการเฉพาะมาตราใดมาตราหนึ่ง หรือภาพรวมทั้งหมด

### ขั้นตอน 2: ค้นหาผ่าน MCP Server

ส่ง request ไปยัง ~~legal-th:
- ค้นหา full-text ในฐานข้อมูลกฎหมาย
- กรองตามหมวดหมู่ถ้าระบุ
- ดึงมาตราที่เกี่ยวข้อง

### ขั้นตอน 3: จัดรูปแบบผลลัพธ์

## หมวดหมู่ที่รองรับ

| หมวดหมู่ | ตัวอย่างกฎหมาย |
|---------|---------------|
| `constitutional` | รัฐธรรมนูญ กฎหมายปกครอง |
| `civil` | ประมวลกฎหมายแพ่งและพาณิชย์ |
| `criminal` | ประมวลกฎหมายอาญา |
| `labor` | พ.ร.บ.คุ้มครองแรงงาน แรงงานสัมพันธ์ |
| `data_protection` | PDPA ระเบียบ สคส. |
| `ip` | ลิขสิทธิ์ สิทธิบัตร เครื่องหมายการค้า |
| `tax` | ประมวลรัษฎากร ภาษีสรรพสามิต |
| `consumer` | คุ้มครองผู้บริโภค |
| `ecommerce` | ธุรกรรมอิเล็กทรอนิกส์ |
| `corporate` | บริษัทมหาชน หลักทรัพย์ |
| `environment` | สิ่งแวดล้อม โรงงาน |
| `healthcare` | สถานพยาบาล วิชาชีพเวชกรรม |

## หมายเหตุ

- ผลการค้นหาอาจมีความล่าช้า 24-48 ชั่วโมงจากการประกาศอย่างเป็นทางการ
- แนะนำให้ตรวจสอบกับราชกิจจานุเบกษาสำหรับความถูกต้องสมบูรณ์
- กฎหมายบางฉบับอาจมีฉบับแก้ไขล่าสุดที่ยังไม่อัปเดตในฐานข้อมูล
