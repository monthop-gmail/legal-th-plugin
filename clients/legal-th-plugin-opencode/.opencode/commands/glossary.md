# /glossary -- ศัพท์กฎหมายไทย

> หากเห็น placeholder ที่ไม่คุ้นเคย ดู [CONNECTORS.md](../CONNECTORS.md)

ค้นหาศัพท์กฎหมายไทย-อังกฤษ พร้อมคำอธิบาย บริบทการใช้งาน และอ้างอิงกฎหมาย

## Usage

```
/glossary $ARGUMENTS
```

ตัวอย่าง:
- `/glossary นิติกรรม`
- `/glossary force majeure`
- `/glossary เหตุสุดวิสัย`
- `/glossary ผู้ควบคุมข้อมูลส่วนบุคคล`
- `/glossary consideration`

## Workflow

### ขั้นตอน 1: รับคำค้นหา

รับศัพท์จากผู้ใช้ ค้นหาได้ทั้งภาษาไทยและอังกฤษ

### ขั้นตอน 2: ค้นหาผ่าน MCP Server

ส่ง request ไปยัง ~~legal-th เพื่อค้นหาใน glossary database

### ขั้นตอน 3: แสดงผล

## Output Format

```markdown
## [ศัพท์ภาษาไทย] / [English Term]

**หมวด**: [สาขากฎหมาย]
**ภาษาไทย**: [คำศัพท์ภาษาไทย]
**English**: [English term]

### คำอธิบาย
[คำอธิบายความหมายทางกฎหมาย]

### บริบทการใช้งาน
[ใช้ในบริบทไหน ตัวอย่างประโยค]

### อ้างอิงกฎหมาย
[มาตรา/กฎหมายที่เกี่ยวข้อง]

### คำที่เกี่ยวข้อง
[ศัพท์ที่เกี่ยวข้องหรือคล้ายกัน]
```

## หมายเหตุ

- ศัพท์บางคำมีความหมายต่างกันในบริบทกฎหมายต่างสาขา
- คำแปลอาจไม่ตรง 1:1 ระหว่างไทย-อังกฤษ ให้ดูคำอธิบายประกอบ
- แนะนำให้ใช้คำศัพท์ตามที่กฎหมายบัญญัติเมื่อร่างเอกสารทางกฎหมาย
